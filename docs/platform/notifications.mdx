## Telegram Setup

OrdsBot sends real-time alerts via Telegram. Setup takes about 2 minutes.

### 1. Create a Telegram bot

Message [@BotFather](https://t.me/BotFather) on Telegram:

```
/newbot
```

Follow the prompts to name your bot. BotFather will give you a token like:
```
1234567890:AAFXuxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### 2. Get your Chat ID

1. Send any message to your new bot
2. Visit: `https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates`
3. Find `"chat":{"id":XXXXXXXXX}` in the response — that number is your Chat ID

For group chats, add the bot to the group first, then send a message and check the same URL.

### 3. Add to OrdsBot Settings

Go to **Settings → Notifications** and enter:
- **Bot Token**: `1234567890:AAFXuxx...`
- **Chat ID**: `-100123456789` (note: group chat IDs start with `-100`)

## Notification Types

### Bid Filled

Sent when a collection bid is accepted by a seller on Satflow.

```
✅ Bid filled
Collection: Bitcoin Puppets
Price: 85,000 sats
Inscription: abc123...
```

### Floor Crash

Sent when OrdsBot detects a floor crash (>15% drop) and cancels bids.

```
⚠️ Floor crash detected
Collection: NodeMonkes
Previous floor: 200,000 sats
Current floor:  165,000 sats (-17.5%)
All bids cancelled.
```

### Task Error

Sent when a worker cycle fails due to an API error, auth issue, or unexpected state.

### Listing Live

Sent when an auto-list task successfully creates a listing on Satflow.
